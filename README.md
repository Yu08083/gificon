# GIF Compression Tool (for Discord Optimization)

Discordのアイコンやバナーに最適な、**8MB以下**に自動調整するブラウザベースのGIF圧縮ツールです。

## 概要
Discord（非Nitroユーザー）のファイルアップロード制限である8MBをターゲットに、GIFの解像度と品質を自動で計算・圧縮します。
サーバーへのアップロードを行わず、すべての処理をブラウザ上のJavaScriptで実行するため、プライバシー面でも安全です。

## 主な機能
- **自動リサイズ計算**: 元のファイルサイズが8MBを超えている場合、目標サイズ（約7.2MB）に収まるようスケールを自動調整。
- **ブラウザ完結型**: `gif.js` と `omggif` を使用し、クライアントサイドでエンコード。
- **プレビュー比較**: 元のGIFと圧縮後のGIFを並べて確認可能。
- **プログレス表示**: 変換の進捗を％でリアルタイム表示。
